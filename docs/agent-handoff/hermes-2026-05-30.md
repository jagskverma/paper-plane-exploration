## What Changed

### 1. Scenery Asset Catalog (SceneryCatalog.ts)
- Expanded from 32 → 218 active assets (205 trees + 13 SimpleNature)

- All scales at 4x multiplier (DB-computed × 4) to match visual tuning iteration

### 2. Collision: Sphere → OBB (FlightController.ts + SceneryGenerator.ts)
- FlightCollisionObstacle now supports `kind: "box"` with `extents: Vector3` and `boxRotation: Quaternion`
- Point-in-OBB algorithm in `closestObstaclePoint`:
  1. Transform plane position into box local space (inverse rotation)
  2. Clamp to [-extent, +extent] on each axis
  3. Transform back to world space
- SceneryGenerator.collisionForPlacement emits box obstacles for all placements
- Box uses collisionRadius×scale for X/Z extents, collisionHeight×scale for Y extent
- Model rotation preserved via placement.rotation

### 4. GLB Database (docs/assets/glb-database.json)

- Used to compute collision dimensions and scale factors
