# Paper Plane Exploration

A browser-first atmospheric flying exploration experience centered around a stylized lowpoly paper plane traversing a large spherical world.

## The Feeling

> drifting through a vast beautiful world

Relaxing. Dreamlike. Atmospheric. Emotionally immersive.

Not a flight sim. Not a combat game. Not a survival sandbox.

## Stack

- Three.js + React Three Fiber
- TypeScript
- Vite
- Browser-native, client-side only

## Quick Start

```bash
cd apps/web
pnpm install
pnpm dev
```
